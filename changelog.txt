Omni-Hub Changelog
v4.5.1 (Bugfix Patch)

    Grid-Logic: Behebung des Layout-Glitch beim dynamischen Hinzufügen/Entfernen von Quick Actions.

    Memory Management: Implementierung von deleteLater() für entfernte Widgets zur Vermeidung von Speicherlecks.

v4.5 (Global Context Update)

    Global Drop-Hub: Umstellung auf persistenten Dateispeicher (Dateien bleiben gelistet).

    Context-Linking: Automatische Dateiverknüpfung zwischen Drop-Hub und Quick Actions/Modulen.

    UI: Hinzufügen eines Slide-out Menüs für erweiterte Datei-Aktionen im Hub.

    Feature: Implementierung eines "Clear List" Buttons im Dashboard-Widget.

v4.4 (Image Pro Module)

    Layout: 3-Spalten-Design (Dateiliste | Action-Center | Output-Vorschau).

    Konsole: Integration einer Status-Konsole für Echtzeit-Feedback der Bearbeitungsschritte.

    Smart Tools: Integration von rembg (Hintergrundentfernung) und Smart Focus Crop.

    UI: Dynamische Parameter-Menüs (z.B. Qualitäts-Slider) je nach gewählter Funktion.

v4.3 / v4.3.1 (Dashboard Refactoring)

    UI-Struktur: Einführung einer vertikalen Sidebar (1x8 Grid) für Quick Actions über die volle Fensterhöhe.

    Visuals: Implementierung von Glassmorphismus-Elementen (1px Rahmen, Transparenz, border-radius: 20px).

    Main Tools: Erweiterung auf 4 Slots mit benutzerdefinierter Favoriten-Auswahl.

    Optimierung: Vertikale Kompression von Drop-Hub und App-Launcher zur Erhöhung der Tool-Listen-Sichtbarkeit.

v4.2 (Quick Action Engine)

    Architektur: Umstellung auf ein modulares Klassen-System für Quick Actions.

    Bibliothek: Integration von 20+ vordefinierten Aktionen (System, Media, AI).

    Customization: Implementierung eines Bibliotheks-Managers zum Hinzufügen/Entfernen von Dashboard-Kacheln.

v4.1 / v4.1.2 (AI Assistant Polish)

    Chat-UI: Umstellung auf Messenger-Layout (User: rechts / AI: links).

    Engine-Switch: Auswahlmöglichkeit zwischen Gemini Pro und Gemini Flash.

    Fallback-Logik: Automatischer Wechsel auf Flash-Modell bei Quota-Überschreitung (Error 429).

    Context Fix: X-Button zum manuellen Entfernen von Datei-Kontexten hinzugefügt.

v3.9.6 (PDF Deep Analysis)

    Feature: Integration von genai.upload_file für die Inhaltsanalyse von PDF-Dokumenten.

    UI: Header-Verkleinerung im Assistant-Modul zur Optimierung der Chat-Fläche.

v3.9.5 (AI Foundation Update)

    API-Integration: Erstimplementierung der Google Gemini API Anbindung.

    Context Engine: Entwicklung des ersten Systems zur Erkennung von Datei-Metadaten für den Chat-Kontext.

    Quick Actions: Einführung der ersten Basis-Aktionen (Zusammenfassen, Erklären) für Textdateien.

v3.5 – v3.9 (Feature Expansion)

    Global Drop-Hub: Einführung des ersten Drag-and-Drop-Bereichs im Dashboard.

    Watchdog System: Implementierung des Hintergrund-Dienstes zur Überwachung von Systemressourcen.

    Tool-Archiv: Aufbau der durchsuchbaren Liste für alle integrierten Sub-Module.

    Design: Umstellung des gesamten Farbschemas auf "Gletscher-Blau" mit abgerundeten UI-Komponenten.

v3.0 (The Dashboard Core)

    Modularer Aufbau: Kompletter Rewrite der GUI-Logik auf ein modulares System für leichtere Erweiterbarkeit.

    Dashboard: Einführung des zentralen Hauptfensters mit Live-Status-Widgets.

    App Launcher: Integration der ersten Schnellstart-Leiste für externe Anwendungen.

v2.0 – v2.5 (Infrastructure & Logic)

    Backend: Migration der Core-Logik auf eine stabilere Python-Klassenstruktur.

    Config System: Implementierung der config.json zur dauerhaften Speicherung von Nutzereinstellungen.

    Local Tools: Integration erster lokaler Werkzeuge wie einfache Dateimanipulation und System-Infoboxen.

v1.0 (Initial Release)

    MVP: Erstellung des Grundgerüsts in PyQt6.

    Basic UI: Erstes Fenster-Layout mit Titelleiste und Navigations-Dummy.

    Base Engine: Grundlegende Initialisierungs-Routinen für die App-Suite.
